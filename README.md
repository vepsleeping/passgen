# PassGen - Генератор случайных паролей

## Описание

PassGen — это консольная программа для генерации надёжных случайных паролей на основе задаваемых пользователем критериев сложности.

## Возможности

- Генерация паролей заданной длины
- Указание количества символов каждой категории:
  - Строчные латинские буквы (a-z)
  - Заглавные латинские буквы (A-Z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*()_-+=<>?/.,;:[]{}|)
- Валидация входных параметров
- Проверка сгенерированного пароля на соответствие критериям
- Использование криптографически стойкой генерации (опционально)

## Требования

- Python 3.9 или выше
